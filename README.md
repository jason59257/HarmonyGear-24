# 优惠券网站 - 前台规划

## 项目结构

```
coupon-website/
├── index.html              # 首页
├── stores.html             # 商店列表页
├── coupons.html            # 优惠券列表页
├── coupon-detail.html      # 优惠券详情页
├── category.html           # 分类浏览页
├── login.html              # 登录页
├── register.html           # 注册页
├── user-dashboard.html     # 用户中心
├── css/
│   ├── styles.css          # 主样式文件
│   ├── components.css      # 组件样式
│   └── responsive.css      # 响应式样式
├── js/
│   ├── main.js             # 主脚本
│   ├── api.js              # API接口
│   └── utils.js            # 工具函数
├── images/                 # 图片资源
└── admin/                  # 后台管理（后续开发）
```

## 功能规划

### 首页功能
- Hero区域（注册推广、现金返还介绍）
- 热门商店展示（带现金返还比例）
- 分类导航
- 最新优惠券
- 统计数据展示
- 浏览器扩展推广
- 移动应用推广

### 商店列表页
- 商店筛选（分类、现金返还比例）
- 商店卡片展示（Logo、名称、返现比例、优惠券数量）
- 搜索功能
- 排序功能

### 优惠券详情页
- 优惠券信息展示
- 复制优惠码功能
- 跳转商店链接
- 现金返还信息
- 使用说明
- 相关优惠券推荐

### 分类浏览页
- 分类列表
- 分类下的商店和优惠券
- 筛选和排序

### 用户功能
- 登录/注册
- 用户中心（我的返现、优惠券收藏）
- 浏览器扩展下载

## 设计特点

参考 Rakuten 的设计风格：
- 现代化的UI设计
- 清晰的现金返还展示
- 响应式布局
- 流畅的用户体验
- 品牌色彩：主色调使用蓝色/绿色（信任感）
